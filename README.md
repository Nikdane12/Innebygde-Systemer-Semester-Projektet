# Innebygde-Systemer-Semester-Projektet
## Sjekkliste
1. Complete the plan of record (POR).
2. Complete the preliminary bill of materials (BOM).
3. Complete the final schematic capture and final BOM.
4. Complete the board layout and order all the parts.
5. Complete the assembly.
6. Complete the bring up, troubleshoot, and final test.
7. Complete the documentation
## POR
- [ ] Visualiser hele prosjeketet før det startes
- [ ] Risikovurdering og risikobegrensing
- [ ] Overordnet beskrivelse av prosjektet, og hva som skal fungere
- [ ] POR er et levende dokument som oppdateres når det kan.
- [ ] Tenk på hva som i verste fall kan skje for hvert steg.
- [ ] Unngå feature creep i denne fasen *
- [ ] Estimer det du kan: lab budsjett for strøm pris og leveringstider
- [ ] BOM for de mest kritiske komponentene
- [ ] Gjør en risikoanalyse, og lag plan B-C osv. For potensielle problemer.
